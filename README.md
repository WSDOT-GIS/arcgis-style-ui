Style UI for ArcGIS feature service
===================================

This project is for designing a UI that will allow the user to change the styling of feature layers in a map.

## Symbols ##

### Simple Line Symbol ###

* style
* color
* width

### Simple Marker Symbol ###

* style
* size
* outline - Line Symbol
    * style
    * color
    * width
* color

### Simple Fill Symbol ###

* style
* outline - Line symbol
    * style
    * color
    * width
* color
